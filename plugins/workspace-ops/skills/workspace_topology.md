# Workspace Topology

- Start with `workspace_status` and `list_registry` so you know what is already loaded.
- Use `workspace-ops:workspace-intel` when you need counts, names, or quick surface search.
- Separate core `.claude` assets from plugin-provided assets so gaps are easier to see.
- Prefer adding to the thinnest high-leverage surface instead of duplicating an existing bundle.
