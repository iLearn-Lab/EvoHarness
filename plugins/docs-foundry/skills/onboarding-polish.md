# Onboarding Polish

- Focus on the first five minutes a user spends with the harness.
- Explain what ships by default and what can be enabled next.
- Prefer one strong onboarding path over a broad wall of options.
- Use the docs surface to remove uncertainty around setup, plugins, and MCP.
