# Release Readiness

- Check the doctor report before assuming the workspace surface is healthy.
- Use recent session and promotion evidence to see whether the workflow has actually held up.
- Prefer naming blockers explicitly over vague caution.
- If release risk is structural, package the fix as a workflow asset instead of another manual reminder.
