# Subagent Orchestration

Use this skill when the task benefits from delegation or parallel inspection.

- assign subagents bounded questions with a clear finish line
- keep tool allowlists narrow and evidence-focused
- prefer exploration, review, or comparison subtasks over open-ended ownership handoff
- ask the subagent for a concise handoff: findings, evidence, and best next step
- if the parent is already stuck in a tool loop, do not spawn another vague loop
- use subagents to reduce context pressure, not to multiply it
