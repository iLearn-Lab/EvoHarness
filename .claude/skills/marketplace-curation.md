# Marketplace Curation

- Rank plugins by daily leverage, not by how many files they contain.
- Avoid shipping three plugins that solve the same job with different names.
- Make install sources stable and obvious so users can recover a bundle quickly.
- Use plugin descriptions to explain the user outcome, not the internal folder structure.
- Review enable/disable behavior so marketplace installs can be toggled cleanly.
